# FileLister
**A simple file browser that shows folder sizes to help identify space-consuming or unneeded files.**

</p>


## Building

```powershell
git clone https://github.com/KOSHAKK/FileLister.git
cd FileLister
mkdir build
cd build
cmake ..
cmake --build .
```
